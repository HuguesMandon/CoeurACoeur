    About Coeur a Coeur

    Coeur a coeur aims to be an open source dating site based around peer-to-peer, using only web pages.

     

    What does this mean? You can review (or even modify, if you so wish) the matching algorithms, the code and everything, to be sure the goal of the site is really to help you find someone, and nothing else. It also means you only need to download the source and launch the CoeurACoeur.html page to have it running, no need for a server. Well, I'm still working on that...
